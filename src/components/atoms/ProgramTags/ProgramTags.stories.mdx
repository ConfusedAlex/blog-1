## Active

You can adjust if the tags are active by passing the `active` prop.

<Preview>
  <Story name="Active">
    <div>
      <ProgramTags active text="docker" />
      <ProgramTags active text="firebase" />

  </Story>
</Preview>

## Text

You can adjust the main text of the component by passing the `text` prop. The colour and background will be determined
by the text typically it will the name of a technology or programming language.

<Preview>
  <Story name="Text">
    <div>
      <ProgramTags text="docker" />
      <ProgramTags text="firebase" />
    </div>
  </Story>
</Preview>
